Importing posts from Blogspot/Blogger

These weeks i"m working on a friend"s webpage, with an embbebed blog. He already has a blog at blogspot.com since some time ago, so told me that could be nice to preserve all blogspot posts in the new blog.

The fact is that the new blog system is done by me, database structure included, so since there"s no converters from blogger/blogspot to another format (another but Wordpress, which doesn"t help) I have to dive into the xml code from blogger export, use some .php, and call it a day!

Information for curious specimens (and php coders):

1. Export blog:
From blogger"s blog configuration, at Basic Tab, click on Export Blog and save the .xml file.

2. Remove namespaces:
Open the .xml file in a good text editor (?Vim, Emacs, TextMate, TextPad...) and remove all thr: namespaces (search/replace the string "thr:" by nothing).

3. Chop the pork:
Then locate the first <entry> node whose <id> node has the string "post", not the string "settings"

Example: <id>tag:blogger.com,1999:blog-18947300.post-113812357977462718</id>

You can do it by searching the string "<entry" and then search for the string ".post".

Once you located this first <entry> node that is a post, remove all text before it.

4. Save the file (with another name better, isn"t it?)

5. Run import_blogger.php from your server, with the file in the same directory.
Change $file variable to point at your new xml file first. When you access to this php it will retrieve you a file called posts_blogger.sql, with the statements required to feed your sql database. You can easily change the schema by editing the .php, lines 20 to 30.


Note: this procedure only retrieves post title, date and text, and no comments (they were dispensable for the new blog).
Note2: This probably doesn"t work for you, but at least it worked for me :P If I can help you with this, drop me a line.
