Photoshop asset generator: How to remove default subfolder when generating image assets

Go to folder:

[Photoshop Folder]\Required\Plug-ins\Generator\assets.generate\lib

We need to edit file filemanager.js. Probably you won"t have write permission to do it, so in Windows right click on it, select Properties, go to the Security tab, click on Edit..., select "Users" and turn on the checkbox under "Allow" and "Write". Click Ok and edit file (if you are on macOS you have permission settings on cmd+i)

go to line 273, where it says:

basePath = path.resolve(documentDirectory, documentName + "-assets");

and replace it for:

basePath = path.resolve(documentDirectory);

Done. From now on, assets will be saved in the same folder as the PSD.
