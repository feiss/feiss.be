AverageMaskToProperty v1.0 for After Effects

Making effects for real image footage in After Effects (CS3) very often requires the creation of masks for showing/hiding parts of the image, and usually you have to rotoscopy that mask to fit it with the footage.

This is not possible with CS3 and arbitrary masks (and you have to do it by hand), but thanks to the Rototracking palette from Alejandro PÃ©rez and friends this can be made in some few and easy steps.

With some specific effect, after doing this, I wanted to put an effect with the origin in the center of the mask. I don"t know if this can be done in an easier way, maybe using expressions, but well, it was a perfect opportunity to learn scripting in AE.

AverageMaskToProperty is the result. You first select a mask, then an arbitrary 2-dimensional Property (a point, a vertex..), and run the script (File/Scripts). You can choose whether to apply to the current frame or to all frames from current to end.

The script calculates the average of the mask vertices and applies the result to the selected property.

I don"t know if you understood anything, but it worked great for me :D
