Working with large files in Photoshop

Sigo vivo eh? :) pero ocupado.. en breve estaré blogerosamente de vuelta. Mientras, algo un poco técnico que a algún guiri puede serle de ayuda.

Hi!, Here are some tips for working with large files (and I am talking of 10,000px, 20,000px.. but could be any that makes your computer scream) in Photoshop. These are based on my own experience and may be not very technical, but for me it works :)
  1. Memory

    This is an easy one: The more memory you have, the better you will work. However, do not hesitate to load your computer with 6 or 8 MB of RAM, because Photoshop only handles a maximum of 2GB (3GB with XP64bits). With more memory the operative system and other programs will run better though.

    The memory reserved by PS at startup can be changed in Edit/Preferences/Performance, but the default setting is normally the best value. Do not put it at 100% with less than 4GB of RAM is a very bad idea, some memory has to be free for the Operative System.

    Look at Adobe for detailed information (Mac systems here).


  2. Settings in Edit/Preferences/Performance


    Here you can lower the History States (10, for example), and the Cache Levels (2, for example). Cache levels are mainly used for zooming out/in, so it will be slower when zooming, but it will consume less memory.

    Also, choose an Scratch Disc partition with lot of free space, with low desfragmentation and good speed. If you have it set to your 1GB-free C:, thats bad thing...


  3. The Master Layer

    Put all layers inside a master layer, and HIDE it before saving the file. This way Photoshop will not save a flattened version of the whole image inside the PSD, and you will reduce A LOT the saving time and the PSD size in disk.

    This is a very time-disk saver tip, but remember that when opening the file in any image viewer it will not show anything, because they use this inexistent flattened version.


  4. Brush spacing


    If you experience slow performance when painting, increase the Brush Spacing parameter. It will draw more separated strokes, but the performance will be very increased.


  5. Crop it!

    This is a little odd but nice tip. When you are working in only a relative small area of your image, hide the other portion of it. This is done by cropping the image but using the parameter "hide" instead of the default one "delete":

    This will result in an apparently cropped image, but do not panic, the rest of your image is still there (try to move a layer).

    The good news is that the performance of Photoshop will increase a lot, and you will work more confortable. The bad news is that you have to re-crop it to revert to the right sized version. This is done again by cropping it with hide option, but cropping to the right dimensions. Since you have lost them, a good way to do it is to save the document limits by setting 4 guides, before cropping it down:


    Now you can crop it back snapping the selection to those guides.


  6. Other tips:

    • Deactivate thumbnails in layers window. They are cute, but PS has to refresh everytime you make a change

    • Hide All Layer Effects. This is a great performance penalti. If you can work with this layers hidden or with its effect hidden, do it.

    • Hide Color Correction Layers. Another important performance penalti. Hide them if you can.
