Más imágenes Donkey Xote // More DonkeyXote images

He subido 4 imágenes más a la web, esta vez de un par de props y varios mattepaintings/texturas para sets que fueron mapeados con la cámara del plano sobre la geometría original del set.

Esta solución permite tener acabados con más detalle y variedad con menos tiempo y coste, pero tiene el inconveniente que sirve para uno (o muy pocos) planos, no se puede aplicar a todas las secuencias porque entonces el coste sí sería enorme.

En Donkey Xote fui el principal mattepainter de sets y "tecnico de camera mappings", y en total habré hecho unos.. 60 o 70 planos de este modo.


I"ve uploaded 4 more pictures to my website, this time one with a couple of props and the rest with some set mattepaintings/textures that were mapped with the shot"s camera over the original set"s geometry.

This solution makes possible to add fine details and variety with less time and budget, but has the inconvenience that it works only for one (or just a few) shots, if it was used for all the secuences the cost would be huge.
