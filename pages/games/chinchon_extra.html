El Chinchón

Cards game for iOS, Android, Windows, Mac and Linux *share_icons*

The game was programmed in AS3 using Starling. This allowed me to have a single codebase for all platforms, to maintain a small binary size and to achieve great performance using atlas textures and accelerated graphics using the GPU. All images and cards were done in Photoshop.

All cards were specifically designed for the game, painted at 300px wide. Final resolution is 114px, and 53px for non-retina iPhones
