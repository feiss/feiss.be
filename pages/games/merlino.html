A Aventura Máxica de Merliño

Graphics and programming, Flash adventure game (2003) *share_icons*

Created in 2001 for the Galician Government, 70,000 cd-rom copies distributed. I made all the artwork and animations, as well as the code for path finding and the dialog system (an incorporated editor was included in the game).

Additional code (main loop, game logic, packaging): David R. Varcarcel
Coordination, story and dialogs: J. Luis Martinez Orge
